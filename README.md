# StyleGuideBot

A RAG-based chatbot that answers style guide questions with precise citations.

## Tech Stack

**Backend:**
- FastAPI
- LangChain
- OpenAI API (GPT-4 + embeddings)
- Chroma Vector DB

**Frontend:**
- React (Vite)
- TailwindCSS

**Deployment:**
- AWS Lambda (backend)
- AWS S3 + CloudFront (frontend)
- AWS API Gateway

## Features

- Upload style guide documents (starting with Wikipedia Manual of Style)
- Natural language queries
- Precise citations from source material
- Deployed on AWS with serverless architecture
